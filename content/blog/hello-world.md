# Hello World

Welcome to the dbohltech blog. This is where I'll share thoughts on development, design, and building things on the internet.

## What to Expect

I'll keep this practical. Posts will be things I actually found useful or interesting while building — not theoretical takes on tech trends.

- Technical write-ups on projects and tools I'm using
- Opinions on frameworks, workflows, and decisions
- Behind-the-scenes on client work (where I can share)
- Occasional design process breakdowns

## How to Add a Post

Drop a new `.md` file in `content/blog/` with frontmatter:

```markdown
---
title: "Your Post Title"
date: "2025-02-01"
excerpt: "A short description that shows on the listing page."
tags: ["tag1", "tag2"]
---

Your content here...
```

That's it — the page picks it up automatically.
